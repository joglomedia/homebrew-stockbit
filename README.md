# homebrew-stockbit

Stockbit Desktop App

Pengalaman Trading Lebih Profesional dengan Aplikasi Saham versi Desktop untuk MacOS

## Installation

Installing Stockbit

```bash
brew tap joglomedia/stockbit
brew install --cask joglomedia/stockbit/stockbit
```

Removing Stockbit

```bash
brew remove --cask joglomedia/stockbit/stockbit
```
